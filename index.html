Questions fréquentes

Vous avez des
questions ?

Toutes les réponses avant de nous contacter.

Nous écrire →
Est-ce que je dois avoir des compétences techniques ?

Aucune. L'espace admin est conçu pour être aussi simple que remplir un formulaire. Vous cliquez, vous modifiez, vous sauvegardez. Une vidéo tutoriel est incluse à la livraison.

Le délai de 7 jours est-il vraiment garanti ?

Oui, contractuellement. Le délai démarre dès réception de vos contenus via notre formulaire d'onboarding. Délai moyen constaté : 5 jours ouvrés.

Que se passe-t-il si je résilie l'abonnement ?

Le site vous appartient intégralement dès le paiement du setup. Si vous résiliez, vous récupérez tous les fichiers. Aucune donnée retenue.

Mon site sera-t-il visible sur Google ?

Votre site est techniquement optimisé pour le référencement dès la mise en ligne. Nous proposons aussi une optimisation de votre fiche Google Business en option.

J'ai déjà un nom de domaine, puis-je le garder ?

Oui. Nous connectons ou transférons votre domaine existant. Votre adresse web reste identique pour vos clients.
